# IULKFF_parkol-
Feladat leírása:
Parkoló nyilvántartó program. A program megmutatja a parkolóház férőhelyét. Belehet Írni, hogy az adott parkolóhelyhez az autó rendszámát. A parkolás gombbal lementi az érkezési időt, a rendszámot, és hogy melyik helyre parkolt le az autó. A távozás gombbal lehet felszabadítani a helyet. Ekkor felugrik egy fizetés ablak ahol lehet fizetni.
